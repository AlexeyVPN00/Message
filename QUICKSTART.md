## 🚀 Запуск (если что-то не работает):

### 1. Запустите Docker:
```bash
docker-compose up -d
```

### 2. Запустите Backend (в одном терминале):
```bash
cd backend
npm run dev
```

Должны увидеть:
```
✅ Database connection established
✅ Socket.IO initialized
🚀 Server running on http://localhost:5000
```

### 3. Запустите Frontend (в другом терминале):
```bash
cd frontend
npm run dev
```

### 4. Откройте браузер:
```
http://localhost:3000
```

### 5. Войдите в систему:
- Email: `user1@test.com`
- Пароль: `user123`

## 🎮 Что можно протестировать:

### 1. Создать чат с другим пользователем:
- В разделе "Диалоги" используйте поиск
- Введите `@user2` или `@admin`
- Кликните на пользователя
- Начните переписку

### 2. Создать групповой чат:
- В разделе "Диалоги" нажмите кнопку "+"
- Введите название группы
- Выберите участников
- Создайте групповой чат

### 3. Добавить в контакты:
- Найдите пользователя через поиск
- Кликните на его аватар/имя чтобы открыть профиль
- Нажмите "Добавить в контакты"
- Перейдите в раздел "Контакты"

### 4. Редактировать профиль:
- Перейдите в раздел "Профиль"
- Нажмите "Редактировать профиль"
- Измените данные и загрузите аватар
- Сохраните изменения

### 5. Переключить тему:
- Внизу бокового меню есть кнопка переключения темы 🌙/☀️

## 🔧 Решение проблем:

### Не могу войти:
1. Проверьте, что backend запущен
2. Используйте email из списка выше
3. Попробуйте зарегистрировать новый аккаунт

### Сообщения не отправляются:
1. Откройте консоль браузера (F12)
2. Проверьте подключение к WebSocket
3. Перезапустите backend

### База данных не работает:
```bash
docker-compose restart
```
